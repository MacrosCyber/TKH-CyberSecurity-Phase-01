# Week 9: Web Application Security
Operating at Layer 7, this module examined the exploitation and remediation of OWASP Top 10 web vulnerabilities. I executed SQL Injection (SQLi) using tautology payloads like `' OR 1=1 --` to bypass authentication mechanisms and exfiltrate database records via UNION attacks. I also deployed Cross-Site Scripting (XSS) payloads (`<script>alert(document.cookie)</script>`) to demonstrate client-side session hijacking. In **TLAB-09: Operation Omni-Portal**, I successfully exploited Broken Object Level Authorization (BOLA), SQLi, and XSS, and subsequently documented the engineering fixes (Parameterized Queries and Output Encoding) required to secure the application code.
