# Week 4: Virtualization & Containers
This module explored environment isolation and containerized infrastructure. I utilized `docker run` to deploy disposable web servers and mapped host ports to container ports to safely expose services. Progressing to orchestration, I implemented `docker-compose up -d` to spin up multi-tier architectures. A crucial security mechanism employed was the `internal: true` flag within Docker Compose networks, which mathematically air-gapped backend databases from the public internet. During **TLAB-04: Operation Fortified Node**, I successfully evicted a rogue container and deployed a secure WordPress/MariaDB stack with strict network segmentation.
