# Week 5: Identity, Access Management & Windows Infrastructure
Focus shifted to enterprise-level access control via Active Directory and Windows Server Core. The core objective was understanding how Identity acts as the modern security perimeter. While operating predominantly in graphical and abstracted cloud environments (GCP), I analyzed the mechanics of Domain Controllers, Organizational Units (OUs), and Group Policy Objects (GPOs) to enforce security hardening across a fleet of endpoints. This theoretical and practical knowledge was tested during **TLAB-05: Operation Unified Front**, which required auditing an Active Directory realm join and verifying the integrity of the corporate identity matrix.
