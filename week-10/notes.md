# Week 10: Digital Forensics & Incident Response (DFIR)
I operated as a Blue Team analyst to conduct a digital autopsy on a compromised system. Using the Sleuth Kit, I executed `fls` to scan disk image indices for orphaned inodes, and `icat` to extract raw payload data, proving that the malware was executing purely in volatile memory. I also deployed an ELK stack, utilizing Kibana Query Language (KQL) to correlate Windows Event IDs and firewall logs. This forensic methodology was applied in **TLAB-10: Operation Phantom Pursuit**, where I successfully reconstructed an adversary's complete kill chain, from initial web server compromise to lateral movement and data exfiltration.
