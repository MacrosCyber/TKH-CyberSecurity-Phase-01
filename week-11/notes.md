# Week 11: Active Defense (Firewalls, IDS & EDR)
The final technical module focused on architecting Defense in Depth. I engineered egress drop rules utilizing `iptables -A OUTPUT` to sever outbound Command and Control (C2) communications. To catch obfuscated malware, I wrote custom Suricata IDS signatures for deep packet inspection, searching for specific hex content within HTTP traffic. Finally, I deployed Sysmon EDR policies to monitor endpoint memory for malicious process creation. This overlapping defense matrix was tested in **TLAB-11: Operation Fortress**, ensuring that if an adversary bypassed the network perimeter, the endpoint monitoring would successfully neutralize the threat.
