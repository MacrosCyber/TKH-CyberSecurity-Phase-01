# Week 3: Python Scripting for Security
Security Orchestration, Automation, and Response (SOAR) principles were introduced by leveraging Python to interact with the underlying operating system. I engineered custom scripts utilizing the `socket` library to build a programmatic port scanner, and the `subprocess` module to execute native Linux commands (like `grep`) directly from Python. A vital function used was `json.dump()`, which allowed me to export extracted threat data into a structured, machine-readable format suitable for SIEM ingestion. These automated detection mechanics were finalized in **TLAB-03: Operation Automated Hunt**, where I built an end-to-end incident response pipeline that parsed raw authentication logs to isolate brute-force attackers.
