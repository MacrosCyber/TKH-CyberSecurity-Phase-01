# Week 6: Phase 1 Midterm (The Hardened Outpost)
The midterm served as a comprehensive capstone, requiring the synthesis of networking, Linux administration, and container security under strict time constraints. During the Practical Diagnostic (The Forge), I executed `find / -type f -user root 2>/dev/null` to locate hidden high-privilege logs while suppressing permission errors, and subsequently locked the evidence down using `chmod 444`. In the primary **Midterm Capstone: The Hardened Outpost**, I served as the sole engineer tasked with deploying an air-gapped Docker stack behind a default-deny UFW firewall, overcoming a live physical hypervisor failure by manually purging corrupted Docker cache volumes to restore operability.
